About this project

Ever wondered which countries get the same bike, or how a model looks in different markets? That's exactly what this site answers. It scrapes live data from BMW Motorrad's regional websites every week via GitHub Actions — so you always see real, up-to-date availability and imagery across all markets. Built by a fan, for fans.

Personally obsessed with the GS series — especially the F 450 GS. If you share that love, you're in the right place. This is an unofficial fan project, not affiliated with BMW Group.
