# BMW

Ever wondered which countries get the same BMW Motorrad bike, or how a model looks across different markets?

This project answers exactly that.

It scrapes live data from BMW Motorrad's regional websites every week using GitHub Actions, giving you up-to-date model availability, regional differences, and official imagery — all in one place.

Whether you're comparing trims, exploring global releases, or just browsing bikes across markets, this dataset stays fresh automatically.

Built by a fan, for fans.

Personally obsessed with the GS lineup — especially the F 450 GS — so you'll definitely see that passion reflected here.

⚠️ This is an unofficial fan project and is not affiliated with BMW Group.
